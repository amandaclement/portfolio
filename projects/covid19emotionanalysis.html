title

COVID-19 Emotion Analysis

type

Data Visualization / Web Design

description

An interactive web-based data visualization illustrating the emotional impact of the COVID-19 pandemic. Each dot represents a Tweet posted on March 11th, 2020, the day the WHO declared the COVID-19 outbreak a global pandemic, coloured by emotion.

The visualization invites users to witness the emotions embedded within global discourse, painting a picture of collective sentiment in the face of adversity.

technologies

JavaScript / Node.js / Express.js / MongoDB / p5.js / HTML / CSS
